Reglas

El juego consiste en que uno de los jugadores elija a otro para preguntarle "Verdad o Reto". Este otro jugador deberá elegir si responder una pregunta con la verdad o realizar un reto. Es requisito para este juego decir la verdad o cumplir con el reto asignado.

¿Qué necesito para jugar?

Ser sincero o animarte a cumplir los retos!

Ganador y Perdedor

No se trata de perdedores o ganadores, se trata de sinceros y valientes!
